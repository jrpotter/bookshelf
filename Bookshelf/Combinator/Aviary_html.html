Bookshelf.Combinator.Aviary

A list of birds as defined in To Mock a Mockingbird. Refer to Bookshelf/Combinator/Aviary for implementation examples.

  • Bald Eagle: \(\hat{E}xy_1y_2y_3z_1z_2z_3 = x(y_1y_2y_3)(z_1z_2z_3)\)
  • Becard: \(B_3xyzw = x(y(zw))\)
  • Blackbird: \(B_1xyzw = x(yzw)\)
  • Bluebird: \(Bxyz = x(yz)\)
  • Bunting: \(B_2xyzwv = x(yzwv)\)
  • Cardinal Once Removed: \(C^*xyzw = xywz\)
  • Cardinal: \(Cxyz = xzy\)
  • Converse Warbler: \(W'xy = yxx\)
  • Dickcissel: \(D_1xyzwv = xyz(wv)\)
  • Double Mockingbird: \(M_2xy = xy(xy)\)
  • Dove: \(Dxyzw = xy(zw)\)
  • Dovekie: \(D_2xyzwv = x(yz)(wv)\)
  • Eagle: \(Exyzwv = xy(zwv)\)
  • Finch Once Removed: \(F^*xyzw = xwzy\)
  • Finch: \(Fxyz = zyx\)
  • Goldfinch: \(Gxyzw = xw(yz)\)
  • Hummingbird: \(Hxyz = xyzy\)
  • Identity Bird: \(Ix = x\)
  • Kestrel: \(Kxy = x\)
  • Lark: \(Lxy = x(yy)\)
  • Mockingbird: \(Mx = xx\)
  • Owl: \(Oxy = y(xy)\)
  • Phoenix: \(\Phi xyzw = x(yw)(zw)\)
  • Psi Bird: \(\Psi xyzw = x(yz)(yw)\)
  • Quacky Bird: \(Q_4xyz = z(yx)\)
  • Queer Bird: \(Qxyz = y(xz)\)
  • Quirky Bird: \(Q_3xyz = z(xy)\)
  • Quixotic Bird: \(Q_1xyz = x(zy)\)
  • Quizzical Bird: \(Q_2xyz = y(zx)\)
  • Robin Once Removed: \(R^*xyzw = xzwy\)
  • Robin: \(Rxyz = yzx\)
  • Sage Bird: \(\Theta x = x(\Theta x)\)
  • Starling: \(Sxyz = xz(yz)\)
  • Thrush: \(Txy = yx\)
  • Turing Bird: \(Uxy = y(xxy)\)
  • Vireo Once Removed: \(V^*xyzw = xwyz\)
  • Vireo: \(Vxyz = zxy\)
  • Warbler: \(Wxy = xyy\)
